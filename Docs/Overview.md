# Deck Defence - 프로젝트 오버뷰

## 한 줄 요약

> 영웅을 직접 조작하며, 카드로 유닛을 소환하고 스펠을 사용하여 밀려오는 적을 막아내는 사이드스크롤 덱빌딩 디펜스 게임.

| 게임 | 참고 요소 |

## 핵심 재미 요소

- **덱빌딩**: 스테이지 사이에서 카드를 획득/제거/강화하며 나만의 덱을 구축
- **실시간 전투**: 영웅을 직접 이동시키며 카드로 유닛 소환, 스펠 시전을 실시간으로 수행
- **자원 관리**: 제한된 마나 내에서 최적의 카드 사용 타이밍 판단
- **시너지**: 카드 간 조합과 시너지를 발견하는 재미

## 게임 루프

### 메타 루프 (캠페인 단위)

```
[캠페인 시작] → [스테이지 선택(맵)] → [전투] → [보상(카드/유물 획득)] → ... → [보스] → [캠페인 종료]
```

1. **캠페인 시작**: 초기 덱(기본 카드 세트)과 영웅을 선택
2. **맵 탐색**: 분기형 맵에서 다음 노드 선택 (전투 / 엘리트 / 상점 / 이벤트 / 휴식 / 보스)
3. **전투**: 사이드스크롤 디펜스 전투 수행
4. **보상**: 전투 승리 시 카드 선택, 골드, 유물 등 획득
5. **보스 격파 → 다음 권역** 또는 **영웅 사망 → 캠페인 종료**

### 전투 루프 (스테이지 단위)

```
[전투 시작] → [영웅 배치 + 카드 드로우] → [영웅 이동 + 마나로 카드 사용] → [유닛 자동 전투] → [적 기지 격파 = 승리]
```
