# 적 시스템 기획

## 개요

각 권역에 등장하는 적 유닛의 구성과 역할을 다룬다.
적의 이름과 외형은 권역 테마 확정 후 결정한다.

## 1. 적 역할 태그

스폰 조합의 다양성을 위해 내부적으로 사용하는 분류. 플레이어에게 직접 노출하지 않는다.

| 태그 | 특징 | 플레이어에게 요구하는 것 |
|------|------|------------------------|
| **물량** | 낮은 벨류, 낮은 스탯, 다수 출현 | AoE/광역 카드 |
| **돌격** | 빠른 이동 속도, 낮은 체력 | 빠른 대응, 저코스트 유닛 배치 |
| **탱커** | 높은 체력, 느린 속도 | 고화력 유닛, 지속 딜 |
| **원거리** | 후방에서 공격, 낮은 체력 | 전선 밀기 또는 원거리 아군으로 대응 |
| **지원** | 주변 아군 강화, 직접 전투력 낮음 | 우선 제거 판단 (스펠 저격, 돌격 침투) |

## 2. 권역별 적 설계 방향

### 2.1 필요한 역할 구성

각 권역은 아래 역할을 커버하는 적 유닛을 최소 1종 이상 포함해야 한다.

- **물량 + 돌격**: 초반부터 등장. 기본 전투에 익숙해지는 구간
- **탱커 + 원거리**: 중반에 해금. 단순 물량 대응으로는 부족해지며 전략적 대응 필요
- **지원**: 후반에 해금. 우선순위 판단을 요구 (지원부터 처리? 탱커부터 깎기?)

> 구체적인 적 이름, 외형, 수치는 권역 테마 확정 후 별도 데이터 시트에서 정의한다.

### 2.2 보스

- 보스 노드에서 적 기지 격파 후 등장하는 강력한 단일 유닛
- 1단계(기지전)를 일반 전투와 동일하게 진행한 뒤, 기지 격파 시 통상 스폰이 중단되고 보스가 출현
- 보스 격파 시 권역 클리어
- 기지전 → 보스전 전환 자체가 페이즈 역할을 하므로, 보스 유닛 자체에 별도 페이즈는 두지 않음
- 높은 스탯 + 고유 공격 패턴으로 차별화
- 상세 설계는 테마 확정 후 진행

## 3. 스폰 조합 참고

적 역할 태그는 스폰 시스템에서 조합 다양성을 위해 활용할 수 있다.

- 같은 역할의 적이 연속으로 편중되지 않도록 가중치 조절
- 예: 탱커 + 원거리 조합, 물량 + 지원 조합 등 역할 믹스 유도
