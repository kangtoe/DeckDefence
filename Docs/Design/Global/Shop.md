# 상점 시스템 기획

## 개요

상점 노드에서 골드를 사용하여 카드 구매/제거/강화, 유물 구매를 할 수 있다.

## 관련 문서

- [맵 & 노드](Map.md) - 맵 구조, 노드 타입
- [카드 시스템](../CardDeck/Card.md) - 카드 분류, 등급, 강화
- [유물 시스템](Relic.md) - 유물 등급, 효과
