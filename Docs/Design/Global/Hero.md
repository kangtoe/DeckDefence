# 영웅 시스템 기획

## 개요

플레이어는 캠페인 시작 시 영웅을 선택한다.
영웅마다 고유 능력, 초기 덱, 전용 카드 풀이 다르며 플레이 스타일을 결정짓는다.

영웅은 [유닛](../Combat/Unit.md)의 한 유형으로, 전투 중 **필드에 직접 존재**하며 기존 기지의 역할을 완전히 흡수한다.
자체 공격력은 없으며, **모든 행동(공격, 버프 등)을 스펠 카드로 수행**한다.

## 1. 영웅 구성 요소

| 요소 | 설명 |
|------|------|
| **특성** | 영웅의 플레이 성향 (범용형, 스펠형, 공격형 등) |
| **시작 유물** | 캠페인 시작 시 자동 부여되는 유물. 영웅의 고유 패시브 역할 |
| **초기 덱** | 캠페인 시작 시 제공되는 기본 카드 세트 (특성에 맞는 구성) |
| **전용 카드** | 해당 영웅 선택 시에만 카드 풀에 추가되는 고유 카드 |
| **주문위력** | 스펠 카드의 효과를 스케일링하는 영웅 스탯 |
| **해금 조건** | 첫 번째 영웅은 기본 해금, 이후 영웅은 권역 클리어로 해금 |

---

## 2. 영웅 선택

- 캠페인 시작 시 해금된 영웅 중 1명을 선택
- 선택 화면에서 시작 유물과 초기 덱을 미리 확인 가능
- 해금되지 않은 영웅은 실루엣 + 해금 조건 표시

---

## 3. 전투 중 영웅

### 3.1 필드 존재

- 영웅은 전투 중 필드에 유닛으로 존재한다
- **기지를 대체**하며, 적 유닛의 최종 목표는 영웅이다
- 영웅이 유닛 소환/카드 시전의 기준점이 된다

### 3.2 이동

- 플레이어가 **좌우로 수동 이동** 조작
- 영웅 위치에 따라 카드 시전 범위와 효과가 달라짐

| 영웅 위치 | 효과 |
|-----------|------|
| **전방** | 스펠/버프 범위가 전선에 가까워짐, 피격 위험 증가 |
| **중앙** | 균형잡힌 포지션 |
| **후방** | 안전하지만 전선까지 카드 효과가 닿지 않을 수 있음 |

### 3.3 전투 행동

- 영웅은 **자체 공격력이 없다**
- 모든 공격/버프는 스펠 카드를 사용하여 수행
- 카드 없이는 이동만 가능
- 카드 사용 시 영웅 전용 모션 없이 **이펙트/파티클로만 연출** — 영웅은 전투 중 Idle, 이동, 피격 애니메이션만 사용

### 3.4 HP

- 영웅 HP는 **캠페인 전체에서 유지**되며, 전투에서 받은 피해가 누적된다
- 휴식 노드에서 회복 가능
- **영웅 HP가 0이 되면 캠페인 종료** (패배)
- 적 유닛이 영웅에게 도달하면 영웅에게 데미지

## 4. 주문위력

주문위력은 스펠 카드의 수치 효과를 결정하는 영웅 고유 스탯이다.

### 4.1 스케일링 공식

```
스펠 효과 = N + (주문위력 × X)
```

| 요소 | 설명 |
|------|------|
| **N** | 카드 고유 기본 효과 (주문위력 0일 때의 효과) |
| **X** | 카드별 위력 계수 (카드마다 다름) |

- 기본 효과(N)가 있으므로 주문위력이 낮아도 스펠이 쓸모없지는 않음
- 위력 계수(X)가 높은 카드일수록 주문위력 투자 효율이 높음

### 4.2 주문위력 변동

| 경로 | 지속 | 설명 |
|------|------|------|
| 영웅 기본값 | 영구 | 영웅마다 초기 주문위력이 다름 |
| 유물/이벤트 | 영구 | 캠페인 진행 중 영구 증감 |
| 전투 내 효과 | 일시 | 특정 카드/효과로 전투 중에만 일시 변동 (TODO) |
