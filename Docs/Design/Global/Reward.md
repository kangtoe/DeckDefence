# 보상 시스템 기획

## 개요

전투 승리 후 카드 선택, 골드, 유물 등의 보상을 획득하는 시스템을 다룬다.

## 관련 문서

- [전투 시스템](../Combat/CombatSystem.md) - 전투 승리 조건
- [카드 시스템](../CardDeck/Card.md) - 카드 분류, 등급
- [유물 시스템](Relic.md) - 유물 등급, 효과
- [맵 & 노드](Map.md) - 노드별 보상 구성

---

## 1. 보상 화면

```
┌─────────────────────────────────────────┐
│              전투 승리!                   │
│                                         │
│          골드 획득: +25                   │
│                                         │
│  ┌──────────────────────────────────┐   │
│  │  [유물 아이콘+이름]  [획득]       │   │
│  └──────────────────────────────────┘   │
│                                         │
│      카드를 선택하세요 (1장):             │
│                                         │
│  ┌────────┐  ┌────────┐  ┌────────┐    │
│  │ 카드 A │  │ 카드 B │  │ 카드 C │    │
│  │        │  │        │  │        │    │
│  │        │  │        │  │        │    │
│  └────────┘  └────────┘  └────────┘    │
│                                         │
│              [스킵]                      │
└─────────────────────────────────────────┘
```

- 골드는 자동 획득
- 유물은 [획득] 버튼으로 수동 획득
- 카드는 호버 시 상세 정보 표시, 클릭으로 선택. 1장 선택 시 나머지 선택지 비활성화
- [스킵] 시 유물·카드 미획득 항목을 모두 건너뛰고 다음 노드로 진행

---

## 2. 골드 보상

- 골드는 현재 [심도](Map.md)에 비례하여 차등 지급
- 노드 타입별 기본 골드량이 다르며, 심도가 높을수록 골드 획득량 증가

---

## 3. 카드 보상

- 기본 3장 중 1장 선택 (특정 상황에서 선택지 수 변동 가능)
- 스킵 가능 (카드를 가져가지 않음)
- 카드 보상은 두 종류로 나뉨:

| 종류 | 등장 카드 | 발생 조건 |
|------|-----------|-----------|
| **일반 카드 보상** | 현재 [심도](Map.md)에 비례한 가중치로 [등급](../CardDeck/Card.md) 무작위 결정 |
| **서사 카드 보상** | 모든 카드가 서사(Epic) 등급으로 등장 |

---

## 4. 전투 노드별 보상 구성

| 노드 | 보상 구성 |
|------|-----------|
| **일반 전투** | 골드 + 일반 카드 보상 |
| **엘리트 전투** | 골드 + 유물 + 일반 카드 보상 |
| **보스** | 골드 + 유물 + 서사 카드 보상 |
