# 유물 시스템 기획

## 개요

유물은 캠페인 전체에 걸쳐 지속되는 패시브 효과를 제공한다.
획득한 유물은 화면 상단에 아이콘으로 표시된다.

## 관련 문서

- [맵 & 노드](Map.md) - 유물 획득 경로 (엘리트, 보스, 상점, 이벤트)
- [아키텍처](../../Guide/Architecture.md) - RelicSystem 이벤트 구독 구조

---

## 1. 유물 등급

| 등급 | 획득 경로 |
|------|-----------|
| 일반 | 엘리트 보상, 상점 |
| 희귀 | 보스 보상, 특수 이벤트 |
| 저주 | 이벤트 (부정적 효과, 강력한 보상과 함께) |

---

## 2. 유물 효과 방향

- 마나/드로우 등 전투 파라미터 강화
- 유닛 소환/사망 시 추가 효과
- 경제 관련 (골드, 상점 가격)
- 전투 시작 시 일회성 보너스
- 저주 유물: 강력한 이점 + 페널티 조합
