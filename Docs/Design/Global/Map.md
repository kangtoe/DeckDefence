# 맵 & 노드 시스템 기획

## 개요

캠페인 내에서 플레이어가 탐색하는 분기형 맵과 각 노드의 동작을 다룬다.

---

## 1. 맵 화면 UI

```
┌─────────────────────────────────────────┐
│  권역 1: 침략받는 왕국    [덱 보기]      │
│  영웅 HP: ████████  골드: 150            │
│                                         │
│          [보스]                          │
│         /      \                        │
│      [전투]  [이벤트]                    │
│       |    \   /   \                    │
│    [상점] [전투] [엘리트]               │
│       \     |     /                     │
│        [  휴식  ]                        │
│         /      \                        │
│      [전투]  [이벤트]                    │
│         \      /                        │
│        [현재 ●]                          │
│                                         │
│  [유물 목록]                             │
└─────────────────────────────────────────┘
```

| 요소 | 설명 |
|------|------|
| 노드 아이콘 | 타입별 구분 (검=전투, 해골=엘리트, 가방=상점 등) |
| 현재 위치 | 강조 표시 (발광/펄스 효과) |
| 선택 가능 노드 | 밝게 표시 + 클릭 가능 |
| 방문 완료 노드 | 흐리게 표시 |
| 미방문 노드 | 아이콘만 표시 |

---

## 2. 맵 생성 규칙

- 슬레이 더 스파이어 스타일의 분기형 노드 맵
- 권역은 여러 노드로 구성
- 복수의 경로가 존재하며 경로 간 교차(합류/분기) 허용
- 보스 노드는 권역의 마지막에 고정
- 엘리트, 상점은 권역당 최소 출현 보장
- **심도(Depth)**: 캠페인 전체의 진행도 카운터. 노드 1개 진행 시 +1 (노드 유형 무관). 초기값 1, 권역과 무관하게 누적. 적 스탯 스케일링의 기준

---

## 3. 노드 타입

| 노드             | 설명                                   |
|------------------|----------------------------------------|
| **일반 전투**    | 기본 전투                              |
| **엘리트 전투**  | 전투 변형 효과 2~3개 적용              |
| **보스**         | 권역 보스                              |
| **상점**         | 카드 구매/제거, 유물 구매, 카드 강화   |
| **휴식**         | 영웅 HP 회복 또는 카드 강화 중 선택    |
| **이벤트**       | 랜덤 이벤트. 선택지에 따라 다양한 결과 |

- 전투 노드별 보상 상세는 [보상 시스템](Reward.md) 참고

---

## 4. 골드

- **획득**: 전투 보상, 이벤트
- **용도**: 상점에서 카드 구매, 카드 제거, 카드 강화, 유물 구매

---

## 5. 권역 구성

- 각 권역은 고유 테마, 배경, 적 계열을 가진다 (콘텐츠 단위)
- 적 스탯 배율은 권역이 아닌 **심도**에 의해 결정됨
- 권역별 보스는 높은 스탯 + 고유 공격 패턴으로 차별화

---

## 6. 영구 해금

캠페인을 반복하며 아래 요소가 영구적으로 해금된다.

| 요소 | 설명 |
|------|------|
| 새로운 영웅 | 권역 클리어 시 해금 |
| 카드 풀 확장 | 누적 캠페인 횟수/업적에 따라 보상 풀에 새 카드 추가 |
| 새로운 유물 | 특정 업적 달성 시 유물 풀에 추가 |
| 난이도(디센션) | 캠페인 클리어 시 다음 디센션 해금. 적 강화/제약 누적 |
| 도감 | 조우한 카드/적/유물 정보 자동 기록 |
