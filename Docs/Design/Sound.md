## 1. BGM

| 상황 | 분위기 | 비고 |
|------|--------|------|
| 타이틀 | 웅장하고 신비로운 오케스트라 | 게임 첫인상 |
| 메인 메뉴 | 잔잔한 판타지 | 루프 |
| 맵 탐색 | 탐험적, 가벼운 긴장감 | 권역별 변주 |
| 일반 전투 | 긴장감 있는 전투 음악 | 권역별 테마 |
| 엘리트 전투 | 일반 전투의 강화 버전 | 템포 빠르게 |
| 보스 전투 | 웅장하고 긴박한 보스 테마 | 보스별 고유 |
| 상점 | 느긋하고 친근한 분위기 | |
| 이벤트 | 미스터리/호기심 | |
| 승리 | 짧은 팡파레 | 2~3초 |
| 패배 | 짧은 슬픈 멜로디 | 2~3초 |

## 2. SFX

### 2.1 UI 효과음

| 상황 | 사운드 |
|------|--------|
| 버튼 클릭 | 가벼운 클릭음 |
| 카드 드로우 | 종이 넘기는 소리 |
| 카드 사용 | 마법 시전음 + 카드 특성별 효과음 |
| 카드 선택 (보상) | 빛나는 효과음 |
| 골드 획득 | 동전 소리 |
| 유물 획득 | 특별한 효과음 (반짝임) |
| 스테이지 클리어 | 승리 팡파레 |

### 2.2 전투 효과음

| 상황 | 사운드 |
|------|--------|
| 근접 공격 | 검/도끼 휘두르는 소리 |
| 원거리 공격 | 화살/마법탄 발사음 |
| 피격 | 타격음 (재질별 상이) |
| 유닛 사망 | 짧은 사망 효과음 |
| 보스 스킬 | 크고 임팩트 있는 효과음 |
| 화염 계열 | 불 타오르는 소리 |
| 번개 계열 | 전기 방전음 |
| 힐 | 부드러운 종소리/바람 소리 |
