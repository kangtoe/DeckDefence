# 카드 시스템 기획

## 개요

카드는 전투 중 플레이어의 모든 행동 수단이다.
덱빌딩을 통해 캠페인 내에서 카드를 획득/제거/강화하며 전략을 구축한다.

## 관련 문서

- [카드 데이터 설계](CardData.md) - 카드 내부 데이터 구조
- [덱 시스템](Deck.md) - 마나, 덱 순환, 핸드/드로우
- [유닛 시스템](../Combat/Unit.md) - 유닛 카드로 소환되는 유닛의 스탯/행동
- [전투 시스템](../Combat/CombatSystem.md) - 카드를 사용하는 전투 환경

## 1. 카드 분류

| 분류 | 설명 | 사용 방식 |
|------|------|-----------|
| **유닛 카드** | 유닛을 필드에 소환 | 사용 시 화면 왼쪽 가장자리에서 유닛 출현 |
| **스펠 카드** | 즉시 효과 발동 (공격/회복/버프 등). 수치 효과는 영웅의 [주문위력](../Global/Hero.md)으로 스케일링 | 영웅 위치 기준으로 대상/범위 지정 또는 즉시 발동 |
| **저주 카드** | 부정적 카드. 핸드에 들어와 방해 | 이벤트·엘리트 등에서 덱에 강제 추가. 사용 불가 또는 사용 시 패널티 (카드별 상이) |

---

## 2. 카드 등급

| 등급 | 색상 | 특징 |
|------|------|------|
| 일반(Common)  | 흰색 | 기본적인 효과 |
| 희귀(Rare)    | 파랑 | 고유한 효과, 빌드 핵심 |
| 서사(Epic)    | 금색 | 강력한 고유 효과, 게임 체인저 |
| 저주(Curse)   | 보라 | 부정적 효과. 사용 불가 또는 사용 시 패널티 |

- 노드 타입에 따라 등급별 등장 확률이 달라짐

## 3. 카드 구성 요소

### 3.1 카드 UI

```
┌──────────────────────┐
│ [비용]           [등급] │
│                        │
│     [일러스트]          │
│                        │
├──────────────────────┤
│ [카드 이름]            │
│ [타입: 유닛/스펠/...]  │
├──────────────────────┤
│ [효과 설명 텍스트]     │
└──────────────────────┘
```

| 요소 | 설명 |
|------|------|
| 비용 | 사용에 필요한 마나 (좌상단) |
| 등급 | 테두리 색상으로 표현 |
| 일러스트 | 카드 중앙 이미지 |
| 이름 | 카드 고유 이름 |
| 타입 | 유닛/스펠/저주 |
| 효과 | 카드의 능력 설명 |

- 카드 데이터 요소 상세는 [카드 데이터 설계](CardData.md) 참고

## 4. 카드 강화

### 4.1 강화 방법

- 상점에서 골드로 강화
- 이벤트 보상
- 휴식 노드에서 HP 회복 대신 선택

### 4.2 강화 효과

- **유닛 카드**: HP/ATK 상승 또는 추가 능력
- **스펠 카드**: 데미지/회복량 증가, 범위 확대, 추가 효과

### 4.3 강화 횟수

- 기본적으로 카드는 **1회만** 강화 가능 → 이름 뒤에 **+** 표시 (예: "보병+")
- **잠재력** 키워드가 붙은 카드만 2회 이상 강화 가능 → **+N** 표기 (예: "보병+" → "보병+2" → "보병+3")

## 5. 덱 관리

### 5.1 카드 획득

| 경로 | 설명 |
|------|------|
| 전투 보상 | 여러 장 중 1장 선택 (스킵 가능) |
| 상점 | 골드로 구매 |
| 이벤트 | 특정 선택지에서 획득 |
| 보스 보상 | 희귀/서사 카드 선택 기회 |

### 5.2 카드 제거

- 상점에서 골드 지불
- 특수 이벤트

### 5.3 빌드 아키타입

캠페인 중 카드 선택에 따라 아래 빌드 방향으로 특화할 수 있다.

| 아키타입 | 핵심 | 핵심 루프 |
|---|---|---|
| **스웜** | 저비용 유닛 다수 소환 | 유닛 대량 소환 → 사망 → 마나 회복 → 재소환 |
| **정예** | 고비용 유닛 + 버프 집중 | 강유닛 소환 시 약유닛 카드 추방 → 덱 압축 → 버프 스펠 집중 |
| **스펠캐스터** | 스펠 중심 + 주문위력 투자 | 주문위력 상승 → 모든 스펠 효과 증폭 |

#### 스웜 (Swarm)

- 저비용 유닛을 빠르게 쏟아내 수적 우위로 밀어붙임
- **핵심 메커니즘**: 아군 유닛 사망 시 마나 회복 → 유닛 회전이 빠를수록 마나 효율 상승
- 소환 시 / 사망 시 트리거 카드, 다수 소환 카드로 구성
- 전체 버프 스펠(아군 전체 재생 부여 등)과 시너지

#### 정예 (Elite)

- 강력한 소수 유닛을 소환하고 버프/재생 스펠로 보호하며 전선 유지
- **핵심 메커니즘**: 유닛 카드 사용 시 핸드의 다른 유닛 카드를 추방(없으면 스킵) → 덱 압축 → 버프 스펠 드로우율 상승
- 강화(유닛 효과) 스택을 유닛에 집중하여 개별 유닛이 여러 체 몫을 함
- 불굴(Unbreakable) 키워드로 핵심 유닛 투자를 보존

#### 스펠캐스터 (Spellcaster)

- 유닛으로 전선만 유지하고, 스펠로 직접 데미지/컨트롤
- **핵심 메커니즘**: 영웅의 [주문위력](../Global/Hero.md)에 투자 → 모든 스펠 효과 동시 강화
- 스펠 효과 공식: `N + (주문위력 × X)` — N은 기본 효과, X는 카드별 위력 계수
- 위력 계수가 높은 카드를 선택할수록 주문위력 투자 효율 상승

## 6. 키워드 시스템

키워드는 카드가 가지는 고정 속성이다. 전장 유닛에 부여되는 유닛 효과는 [유닛 시스템 - 유닛 효과](../Combat/Unit.md)를 참고.

| 키워드 | 설명 |
|--------|------|
| **선봉(Vanguard)** | 전투 시작 시 반드시 초기 핸드에 포함 |
| **붕괴(Breakdown)** | 리드로우 시 핸드에 남아 있으면 추방 |
| **추방(Exile)** | 사용 시 버리는 더미가 아닌 전투에서 완전히 제거 |
| **보존(Reserve)** | 리드로우 시 핸드에서 버려지지 않고 유지 |
| **사용불가(Unplayable)** | 핸드에서 직접 사용할 수 없음 |
| **불굴(Unbreakable)** | 유닛 카드 전용. 사용 시 카드 추방 + 해당 유닛 사망 시 핸드에 카드 생성 |
| **잠재력(Potential)** | 이 카드는 2회 이상 강화 가능 |

## 기타 사항

- 카드는 앞/뒷면 구분을 둔다. 앞면에는 기본 카드 정보(비용, 일러스트, 이름, 효과 등), 뒷면에는 세부 정보(상세 스탯, 시너지 등)를 표시할 예정
- 등급별 테두리 색상으로 등급 표현
- 강화된 카드는 테두리에 발광 효과
- 카드 타입별 배경 색조 구분
