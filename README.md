# Deck Defence

카드로 유닛을 소환하고 스펠을 시전하여 밀려오는 적을 막아내는 사이드스크롤 덱빌딩 디펜스 게임.

## 기술 스택

- **엔진**: Unity 2D
- **언어**: C#
- **해상도**: 1920x1080 (16:9)
- **타겟 FPS**: 60
